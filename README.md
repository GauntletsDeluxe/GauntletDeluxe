# Gauntlets Deluxe

**A <cl>community-run</c> custom gauntlet levels.**

---

### About

This mod adds a custom gauntlet to the game picked by the community. Contains **fan-made gauntlets** based of it's theme.

Allows <cl>community-run</c> creator contest/gauntlet events hosted here!

### <co>You can find the Gauntlets Deluxe at the Gauntlets tab in the Creator Page. You will find the button at the top right of the screen in the Gauntlet tab.</c>

---

### Features

- <cp>**Levels Points**</c>: Earn points by completing levels in a gauntlet. Each level points are different not based of difficulty.
- <cr>**Gauntlet Points**</c>: Earn points by completing gauntlets. Each gauntlet points are different based on the amount of levels and difficulty.
- <cg>**Gauntlet Leaderboard**</c>: Compete with other players for the top spot on the leaderboard.
- <cf>**Fan-Made Creator Contest Gauntlets**</c>: Play through gauntlets created by the community as part of the Community-hosted Creator Contest.
- <cb>**Entirely Community-run**</c>: This mod is entirely run by the community, from the creation of gauntlets to the management of the leaderboard.
